# UnscrambleGame

## 1. Название проекта

**UnscrambleGame** — игра «Разгадай слово»

---

## 2. Краткое описание игры

**UnscrambleGame** — это обучающая игра для Android, в которой игроку нужно отгадать исходное слово из перемешанных букв.

**Как играть:**
- На экране отображается слово с перемешанными буквами (например, "ндроиа" вместо "андроид")
- Игрок вводит свой вариант ответа в текстовое поле
- За каждый правильный ответ начисляется 20 очков
- Игра продолжается до тех пор, пока не будет отгадано 10 слов
- Можно пропустить слово — очки не начисляются, но игра продолжается
- В конце игры показывается диалог с итоговым счётом и предложением сыграть снова

**Цель игры:** набрать максимальное количество очков, правильно отгадав все 10 слов.

## 4. Функциональность приложения (список возможностей)

### Основные возможности:

1. **Отображение перемешанного слова**
    - Случайное слово из списка отображается с перемешанными буквами
    - Слово отображается крупным шрифтом в центре экрана

2. **Ввод ответа пользователя**
    - Текстовое поле для ввода отгаданного слова
    - Кнопка "Готово" на клавиатуре для отправки ответа
    - Автоматическая проверка регистра (Android = android = ANDROID)

3. **Проверка правильности ответа**
    - При правильном ответе: начисляется 20 очков, загружается следующее слово
    - При неправильном ответе: появляется красное сообщение об ошибке, счёт не меняется

4. **Отслеживание прогресса**
    - Отображение номера текущего вопроса (например, "Слово 3 из 10")
    - Отображение текущего счёта игрока

5. **Пропуск слова**
    - Возможность пропустить сложное слово
    - Счёт при пропуске не меняется
    - Загружается следующее слово

6. **Завершение игры**
    - После отгадывания 10 слов появляется диалоговое окно
    - Отображается итоговый счёт игрока
    - Кнопка "Играть снова" для перезапуска игры

7. **Сохранение состояния при повороте экрана**
    - Благодаря ViewModel все данные сохраняются
    - Счёт, текущее слово, номер вопроса не сбрасываются

8. **Адаптивный интерфейс**
    - Поддержка портретной и ландшафтной ориентации
    - Прокрутка экрана при необходимости

9. **Модульное тестирование**
    - Тесты для проверки правильных ответов (Happy Path)
    - Тесты для проверки неправильных ответов (Error Path)
    - Тесты для проверки начального состояния (Boundary Case)
    - Тесты для проверки завершения игры
